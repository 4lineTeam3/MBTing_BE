# MBTing_BE

### 💁 server키는 법

‼️ python3,pip3 안될 시 -> python, pip ‼️

# window
  
  초기 설정

  1. 가상환경 생성

    python3 -m venv venv

  2. 가상환경 진입

    source venv/Scripts/activate

  3. requirements 설치

    pip3 install -r requirements.txt

  4. 실행

    python3 manage.py runserver

  이후 실행 시 2~4번만 실행

# mac

  초기 설정

  1. 가상환경 생성

    python3 -m venv venv

  2. 가상환경 진입

    source venv/bin/activate

  3. requirements 설치

    pip3 install -r requirements.txt

  4. 실행

    python3 manage.py runserver

  이후 실행 시 2~4번만 실행
